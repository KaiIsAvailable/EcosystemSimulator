CUSTOMIZATION:
--------------

Position Options:
- Bottom-Left (default): Good for event log
- Top-Left: Alternative if dashboard is on right
- Bottom-Right: Mirror layout

Color Customization (in EventNotificationUI.cs):
- Eating: new Color(0.5f, 1f, 0.5f) - Light green
- Starving: new Color(1f, 0.8f, 0.3f) - Orange
- Death: new Color(1f, 0.3f, 0.3f) - Red
- Grass: new Color(0.7f, 0.9f, 0.7f) - Pale green

TROUBLESHOOTING:
----------------

Q: Notifications don't appear?
A: - Check Rich Text is enabled on Text component
   - Verify EventNotificationUI is on a GameObject
   - Check "Show Notifications" is enabled
   - Look for errors in Console

Q: Colors don't show?
A: - Rich Text MUST be enabled on Text component
   - Check Text component uses Legacy Text (not TextMeshPro)

Q: Too many/few notifications?
A: - Adjust "Max Notifications" (default: 5)
   - Adjust "Notification Duration" (default: 3 seconds)

Q: Notifications fade too fast?
A: - Increase "Notification Duration" to 5-10 seconds
